# Task 6 - Password Strength Evaluation

## Objective
To understand what makes a password strong and evaluate password strength using an online password strength checker.

## Tool Used
- PasswordMeter (https://www.passwordmeter.com)

## Tested Passwords

| Password | Strength |
|-----------|-----------|
| password123 | Weak |
| Welcome2025 | Medium |
| Welcome@2025 | Strong |
| MySecure#Pass2025! | Very Strong |
| Q7@vL9#xP2!mR8$k | Extremely Strong |

## Findings

- Longer passwords are harder to crack.
- Combining uppercase and lowercase letters improves security.
- Including numbers and special characters increases strength.
- Avoid dictionary words and predictable patterns.
- Unique passwords should be used for every account.

## Common Password Attacks

### Brute Force Attack
An attacker tries every possible combination until the correct password is found.

### Dictionary Attack
An attacker uses a list of common words and leaked passwords to guess credentials.

## Best Practices

- Use at least 12-16 characters.
- Include uppercase, lowercase, numbers, and symbols.
- Avoid personal information.
- Use a password manager.
- Enable Multi-Factor Authentication (MFA).

## Conclusion

Strong passwords significantly improve account security and reduce the risk of unauthorized access.
